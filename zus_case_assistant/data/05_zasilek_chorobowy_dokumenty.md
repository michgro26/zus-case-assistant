# Zasiłek chorobowy - wymagane dokumenty
Źródło publiczne: https://www.zus.pl/swiadczenia/zasilki/zasilek-chorobowy/z-ubezpieczenia-chorobowego/niezbedne-dokumenty
Data dostępu: 2026-04-15

Najważniejsze informacje:
- Podstawą do zasiłku chorobowego jest zwolnienie lekarskie e-ZLA, jego wydruk albo formularz z systemu teleinformatycznego, jeśli wystawienie e-ZLA nie było możliwe.
- Gdy zasiłek wypłaca ZUS, potrzebne jest też zaświadczenie płatnika składek na druku Z-3 dla pracownika.
- Dla osoby prowadzącej pozarolniczą działalność, osoby współpracującej, osoby korzystającej z ulgi na start lub duchownego właściwy jest druk Z-3b albo wniosek ZAS-53.
- Jeżeli klient ubiega się o zasiłek chorobowy za okres po ustaniu ubezpieczenia, powinien dodatkowo złożyć oświadczenie na druku Z-10.

Wskazówki do obsługi sprawy:
- W analizie sprawy należy rozróżnić status klienta: pracownik, działalność gospodarcza, okres po ustaniu ubezpieczenia.
- Przy brakach dokumentów warto wskazać dokładnie brakujący formularz, a nie tylko ogólnie „uzupełnić dokumenty”.
